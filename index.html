한국기독교장로회
전국목회자 축구대회 점수표

5개 팀씩 2개 조, 조별 풀리그 결과를 입력하면 승점·득실·경고/퇴장 감점까지 자동 계산합니다. 현장 운영자는 경기 결과만 입력하고, 대시보드는 실시간으로 순위를 정리합니다.

승점-감점 후 총점 기준

전체 통합 순위

조가 달라도 현재 성적을 한눈에 비교합니다.

경기 결과 입력

스코어를 입력하면 승/무/패와 순위가 자동 반영됩니다. 경고 1장 = 0.5점 감점, 퇴장 1명 = 1점 감점.

팀 선택

탭을 누르면 팀별 경기/징계/메모를 봅니다.

A조 팀명

팀명 수정 후 저장하면 경기표에도 반영됩니다.

B조 팀명

기본값은 A1~A5, B1~B5입니다.

순위 계산 규칙

기본 정렬: 총점(승점-감점) → 원승점 → 득실차 → 다득점 → 경고/퇴장 적은 팀 → 팀명. 동률 세부 규정이 별도로 있으면 다음 버전에서 맞춤 적용하면 됩니다.

권장 운영: 공식 기록자 1명이 한 기기에서 입력하고, 중간중간 JSON 내보내기로 백업하세요. 여러 사람이 동시에 입력해야 한다면 Supabase/Firebase 같은 서버 저장 방식으로 확장하는 것이 안전합니다.
