관리 도구

점수/팀/경기 기록은 이제 브라우저에 저장하지 않고 GitHub 파일 data/scoreboard.json만 기준으로 사용합니다. 화면을 열면 온라인 기록을 자동으로 불러오고, 수정 후에는 온라인 덮어쓰기 저장을 눌러야 다른 사람에게 반영됩니다. 토큰만 현재 브라우저에 저장할 수 있습니다.

운영 권장: 화면을 열면 온라인 기록을 불러오고, 수정 후에는 반드시 온라인 덮어쓰기 저장을 눌러야 다른 사람 화면에도 반영됩니다. 동시에 수정하면 마지막 저장이 기준입니다.
