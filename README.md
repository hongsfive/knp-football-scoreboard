# 한국기독교장로회 전국목회자 축구대회 점수표

5개 팀씩 2개 조, 조별 풀리그 예선 점수표를 입력/자동 계산하는 정적 웹앱 프로토타입입니다.

## MVP 기능

- A조: 대전, 전북, 경기남, 충북, 군산
- B조: 익산, 전북동, 충남, 경기중부인천, 경기
- A/B 2개 조, 각 5개 팀 팀명 수정
- 조별 풀리그 10경기씩 자동 생성
- 경기 스코어 입력
- 경고/퇴장 입력
- 승점 자동 계산: 승 3, 무 1, 패 0
- 감점 자동 계산: 경고 1장 = 0.5점, 퇴장 1명 = 1점
- 총점: 승점 - 감점
- 조별 순위 / 전체 통합 순위 자동 계산
- 팀별 기록 탭
- localStorage 자동 저장
- JSON 백업/복구
- CSV/스프레드시트용 순위 복사
- 인쇄/PDF 출력

## 현재 순위 정렬 기준

1. 총점 = 승점 - 감점
2. 원 승점
3. 득실차
4. 다득점
5. 감점 적은 팀
6. 팀명 가나다순

대회 공식 동률 규정이 따로 있으면 이 부분을 바꾸면 됩니다.
