# PROK 한국기독교장로회 전국목회자 축구대회 점수판

A/B/C조 경기 결과를 조별로 입력하고, 메인 대시보드와 팀별 점수판에서 전체 현황을 확인하는 정적 웹앱 프로토타입입니다.

## MVP 기능

- 메인 대시보드: A/B/C조 순위, 1부 통합 순위, 다음 경기/결선 흐름
- 팀별 점수판: 팀별 순위, 승점, 경고, 득실, 남은 경기, 메모
- 조별 입력화면: A조/B조/C조 입력 탭 분리
- 경기 일정: 첨부 이미지 기준 11일/12일 조별 경기 순서 반영
- C조: 경북경남제주강원, 광주남, 서울북, 광주 4개 팀 기준, 2부리그로 별도 계산
- 스코어, 경고, 퇴장 입력
- 승점 자동 계산: 승 3, 무 1, 패 0
- 순위 결정: 승점 → 경고 누적 횟수 → 골득실차 → 다득점 → 승자승 → 제비뽑기
- localStorage 자동 저장
- JSON 백업/복구
- CSV/스프레드시트용 순위 복사
- 인쇄/PDF 출력

## 화면 구성

1. **메인 대시보드**
   - 상단을 가볍게 유지하고 전체 현황이 바로 보이도록 구성
   - A/B/C조 순위와 다음 경기, 결선 흐름 중심
   - A/B조는 1부리그 통합 순위로 비교, C조는 2부리그로 별도 계산
2. **팀별 점수판**
   - 각 팀 카드형 점수판
   - 본부/운영자가 팀 상태를 빠르게 확인
3. **조별 입력화면**
   - A조, B조, C조 입력 탭 분리
   - 각 조 기록자가 자기 조 경기만 입력하기 쉬운 구조
4. **팀명/규칙**
   - 팀명 수정과 순위 계산 규칙 확인
   - 첫 화면을 가볍게 유지하기 위해 저장, JSON 백업/복구, CSV 복사, 인쇄/PDF, 초기화 버튼을 이 화면의 관리 도구로 분리

## 현재 순위 결정 기준

1. 승점
2. 경고 누적 횟수 적은 팀
3. 골득실차
4. 다득점
5. 승자승
6. 제비뽑기

제비뽑기는 앱이 자동 결정하지 않습니다. 승자승까지 모두 같으면 공동 순위로 표시하고, 현장에서 제비뽑기로 최종 결정하면 됩니다.

## 운영 주의

현재 버전은 브라우저 localStorage에 저장합니다. 각 조 담당자가 별도 기기에서 입력하면 서로 자동 동기화되지 않습니다.

- 단일 기기/본부 입력: 현재 버전으로 충분
- 각 조 담당자가 각자 입력하고 본부 화면에 실시간 합산: Supabase/Firebase 같은 서버 저장소 필요
- 운영 중에는 중간중간 JSON 백업을 권장합니다.
